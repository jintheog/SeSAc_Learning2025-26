# 참조 자료형

# 배열

- 0개 이상의 데이터(원소, 요소) 저장
- 배열의 생성 :

```jsx
const arr = [1, 2, 3, 4];
```

- 배열의 반복문 :
  - for문 : 배열의 인덱스를 활용
  - for...of : 배열의 원소에 직접 접근

# 객체

- 0개 이상의 데이터 (속성= 키 + 값) 저장
- 객제의 생성 :

```jsx
const obj = {
  key: "value",
  key1: "value2",
};
```

- 객체의 속성 접근

  - 키(key)를 활용해서 접근

- for ... in : 객체의 key 순회
