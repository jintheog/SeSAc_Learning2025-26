# 객체

- 0개 이상의 속성을 저장 하는 자료형
  속성 = key(이름) + value(값)

- {} 중괄호로 생성

변수 emptyObject를 선언하고
빈 객체를 할당

```jsx
let emptyObject = {};
```

- key는 name
- value는 "홍길동"인 속성

#### 속성 -> key : value

- 변수 person1 할당

```jsx
let person1 = {
  name: "홍길동";
};
```

- 객체 생성 기본구조

```jsx
let 변수명 = {
  key1: value1,
  key2: value2,
  key2: value3,
  key2: value4,
};
```
